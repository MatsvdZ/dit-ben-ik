# dit ben ik
Mijn eerste site voor de minor Web Design & Development


## Eerste checkout (03-02)

Het eerste check-out gesprekje was omdat het de eerste les was, erg kort, maar krachtig. Eigenlijk werd er gevraagd hoe we er nu voor staan en of alles duidelijk is. Dit was gelukkig zo. We kregen als tip mee om gewoon lekker te beginnen, en vooral niet te groot te beginnen. Je kunt altijd later nog uitbreiden als hier tijd voor is.

Ik heb vandaag voor mijzelf duidelijk alles op een rijtje gezet. Ook heb ik meteen al een repo op GitHub aangemaakt en mijn code klaargezet zodat ik er de volgende dag meteen meer door kan gaan. 

Ik heb op dit moment nog niks geleerd, maar daar komt natuurlijk wel verandering in. 

Ik ga morgen gewoon lekker beginnen met mijn project coderen. Ik wil in ieder geval al een sterke basis neerzetten en eventueel alvast beginnen met het kijken naar eventuele animaties. Ook hebben we al meteen de eerste workshops op de planning. Ik ga scroll driven animations volgen, en ik hoop dit ook te kunnen gebruiken voor mijn eigen project.

## Tweede checkout (04-02)

### Wat heb ik gedaan?
We hebben vandaag workshops gehad. Ik heb meegedaan met de workshops over scroll driven animations en de workshop over endpoints/custom JSON.

Ik heb daarna gewerkt aan het werkend maken van de API. Deze is nu werkend op mijn site, en ik kan nu al mijn data inladen.

Ik heb verder ook nog geoefend met keyframes voor mijn eigen project. Hierin heb ik geprobeerd om een image een bepaald pad te laten volgen. Dit is gelukt, maar nog niet hoe ik het zou willen

### Hoe lang duurde het?
De workshops duurde ieder een uurtje, dus daar ben k in totaal twee uur aan kwijt geweest. Na de workshop zelf heb ik effectief drie uur aan mijn eigen project gezeten.

###  Wat heb ik geleerd?
Ik heb een andere manier van API structuur gezien en geleerd. Dit is wel leuk om te zien hoe dit op vele verschillende manieren kan.

Ook heb ik gebruik gemaakt van de workshops door meteen te oefenen met de keyframes, door een image een bepaalde path te laten volgen.

### Wat ga ik morgen doen?
Ik ga morgen beginnen met het verder coderen en oefenen met het laten bewegen van mijn img, en ik ga ook proberen een basis site nee te zetten, zodat ik voor vrijdag iets heb om te laten zien.
